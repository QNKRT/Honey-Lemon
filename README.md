# 🍰 Honey Lemon

Sitio web para la pastelería "Honey Lemon" para acelerar el proceso de cotizar y ordenar pasteles personalizados.
Este proyecto está siendo desarrollado como parte de mi proceso de aprendizaje en desarrollo web.

## 🚀 Descripción

Honey Lemon es una página donde los usuarios pueden ordenar un pastel personalizado seleccionando:

- Forma
- Tamaño
- Sabores de bizcocho
- Rellenos
- Extras

El objetivo es construir una experiencia clara, visual y fácil de usar.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (próximamente Grid y Flexbox)
- JavaScript (en desarrollo)
- Git & GitHub

## 🎯 Objetivo del proyecto

Acelerar el proceso para clientes nuevos y frecuentes de la pastelería.

-Proceso intuitivo
-Opciones claras

Este proyecto forma parte de mi camino para convertirme en desarrollador full stack, enfocado en:

- Estructura semántica
- Buenas prácticas
- Interfaz centrada en el usuario
